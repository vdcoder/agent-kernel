# Data Agent Knowledge Base

Real-Time Data Access Guide

---

# 1. Purpose

The Data Agent provides **real-time access** to guest and ship information maintained by ship systems.

The agent retrieves live information rather than storing static copies.

---

# 2. Guest Profile Data

The Data Agent can retrieve the following guest profile information:

## Personal Information

- Guest name
- Cabin number
- Cabin category
- Loyalty tier
- Loyalty points balance
- Sailing dates
- Ship assignment
- Birthdays and anniversaries
- Dietary restrictions
- Food allergies
- Accessibility needs
- Past cruise history

---

# 3. Reservation Data

The Data Agent can retrieve reservation details for:

| Reservation Type | Available Information |
|------------------|-----------------------|
| Dining | Venue, date, time, party size, confirmation number |
| Spa | Treatment, date, time, therapist, confirmation number |
| Shows | Show, date, time, seats, confirmation number |
| Shore Excursions | Tour, port, date, time, guests, confirmation number |

---

# 4. Account & Folio Data

## Current Account Information

The Data Agent can retrieve:

- Current account balance
- Payment method on file
- Credit limit status

---

## Transaction History

The Data Agent can retrieve:

- Charges (date, time, venue, amount)
- Credits
  - Onboard credits
  - Promotions
- Daily gratuity charges
- Service charges

---

# 5. Ship Schedule Data

## Daily Schedule

The Data Agent can retrieve:

- Port or Sea Day status
- Arrival times
- Departure times
- All Aboard time
- Tender status

---

## Event Schedule

The Data Agent can retrieve:

- Daily activities
- Show schedules
- Restaurant hours
- Venue operating hours

---

# 6. Real-Time Operational Data

| Data Type | Information Available |
|-----------|-----------------------|
| Wait Times | Buffet, Guest Services, Shore Excursions Desk |
| Venue Status | Open/Closed, current capacity |
| Pool Status | Open/Closed, crowding level |
| Tender Status | Running status, wait time, last tender |
| Weather | Current conditions and forecast |

---

# 7. Privacy & Access Control

## Guest Identification

Before accessing guest-specific information, verify:

- Guest name
- Cabin number

---

## Privacy Policy

Guest information may only be shared with:

- The verified guest
- Authorized cabin mates

---

## Sensitive Information

The Data Agent must never display full payment details.

Only the **last four digits** of payment methods may be shown.

---

## Data Retention

Transaction history remains available:

- During the cruise
- For 30 days after the cruise
