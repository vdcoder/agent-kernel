# 3. Dietary Accommodations

| Dietary Need | Availability | Notes |
|--------------|--------------|------|
| Vegetarian | All venues | Multiple options every meal |
| Vegan | All venues | Request vegan menu; 24-hour notice helpful |
| Gluten-Free | All venues | Gluten-free bread and pasta available |
| Kosher | Main Dining Room | Request 90 days before sailing |
| Halal | Main Dining Room | Request 30 days before sailing |
| Nut-Free | All venues | Alert your server; separate preparation available |

## Allergens Tracked

- Peanuts
- Tree nuts
- Milk
- Eggs
- Wheat
- Soy
- Fish
- Shellfish
- Sesame

### Allergy Process

1. Note allergies during booking.
2. Confirm with the Head Waiter during embarkation.
3. Review the menu nightly with your server.
